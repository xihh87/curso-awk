# ¿Qué significa AWK?

AWK es un minilenguaje para el análisis de textos.

- Alfred **A**ho
- Peter **W**einberger
- Brian **K**ernighan

---

# Un pseudo-programa de AWK

```
#!/usr/bin/awk -f
BEGIN { acciones antes de leer la entrada de datos }
condicion { acciones }
END { acciones al final de la lectura de datos }
```
